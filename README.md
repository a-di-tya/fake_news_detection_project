# fake_news_detection_project
-> Short Description / Purpose:In today’s digital world, misinformation spreads very quickly and can influence people’s opinions and decisions. This project aims to tackle that problem by building a Fake News Detection System that can analyze news content and determine whether it is real or fake.
The system uses a combination of machine learning techniques and real-time fact-checking APIs to provide reliable results. The goal is to create a simple yet effective tool that helps users verify the authenticity of news before trusting or sharing it.

->Tech Stack:-
Programming Language: Python

Libraries: Pandas, NumPy, Scikit-learn, NLTK

API Integration: Google Fact Check API

Model Used: Logistic Regression 

Frontend : Flask

Tools:VS Code

Data Source:  Kaggle datasets
Real-time verification using Google Fact Check API.

->Highlights / Features:-

Business Problem:Fake news spreads rapidly on social media and news platforms, making it difficult for users to identify trustworthy information. There is a need for a system that can automatically verify news content.

Goal of the Project:To build a system that can classify news as real or fake
To integrate real-time fact-checking for better accuracy
To provide a user-friendly way to verify news articles

How It Works (Workflow):User inputs a news headline or article
Text preprocessing is applied (cleaning, tokenization, stopword removal)
The trained ML model analyzes the content
The system also fetches related fact-check data using API
Final prediction is shown as Fake or Real.

Key Features:Real-time news verification using API
Machine learning-based classification
Clean and simple interface
Fast and efficient prediction
Helps reduce misinformation spread.

->Business Impact / Insights:Encourages users to verify news before sharing
Helps in reducing the spread of fake information
Can be extended for use in social media platforms or news agencies
Improves awareness about misinformation
